# 体育课堂智能评估系统

基于姿态估计与运动识别的中小学体育课堂智能评估模型构建

## 🎯 项目简介

本项目是一个基于AI姿态估计技术的体育课堂智能评估系统，能够实时检测学生的运动姿态，自动识别运动类型，并提供专业的动作评估和改进建议。

**在线演示**: https://u5aflale74i52.ok.kimi.link

## ✨ 核心功能

### 1. 实时姿态检测
- 基于 MediaPipe Pose 实现高精度人体关键点检测
- 33个关键点追踪，覆盖全身主要关节
- 实时骨骼连线可视化

### 2. 运动类型识别（9种）
支持九种常见体育动作的自动识别：

| 运动类型 | 检测内容 | 难度 |
|---------|---------|------|
| **深蹲** | 膝盖角度和深蹲深度 | ⭐⭐ |
| **俯卧撑** | 肘部角度和身体姿态 | ⭐⭐⭐ |
| **跳跃** | 跳跃高度和膝盖弯曲 | ⭐⭐ |
| **高抬腿** | 抬腿高度和髋关节活动 | ⭐⭐ |
| **开合跳** | 手臂和腿部开合幅度 | ⭐ |
| **仰卧起坐** | 躯干角度和起身幅度 | ⭐⭐ |
| **平板支撑** | 身体挺直度和稳定性 | ⭐⭐⭐ |
| **弓步蹲** | 前后腿膝盖角度 | ⭐⭐ |
| **波比跳** | 多阶段动作连贯性 | ⭐⭐⭐⭐ |

### 3. 智能评估系统
- 动作标准度评分（0-100分）
- 实时反馈和建议
- 动作次数自动计数
- 训练时长统计

### 4. 数据可视化
- 实时评分展示
- 评估历史记录
- 训练数据统计
- 动作改进建议

## 📖 使用说明

1. 打开网页后，点击"开始评估"按钮
2. 允许浏览器访问摄像头
3. 站在摄像头前，保持全身可见
4. 开始运动（深蹲、俯卧撑、跳跃等）
5. 系统将自动识别动作并给出评分和建议

## 🔍 评估算法说明

### 深蹲评估
- **膝盖角度**: 检测双膝弯曲角度
- **深蹲深度**: shallow(浅) / parallel(平行) / deep(深)
- **评分标准**: 深度 + 背部挺直度 + 双膝平衡

### 俯卧撑评估
- **肘部角度**: 检测肘部弯曲程度
- **身体姿态**: 检测身体是否保持挺直
- **评分标准**: 下降深度 + 身体姿态 + 双臂平衡

### 跳跃评估
- **跳跃高度**: 相对基准位置的垂直位移
- **膝盖弯曲**: 起跳时膝盖弯曲程度
- **评分标准**: 跳跃高度 + 起跳姿势

### 高抬腿评估
- **抬腿高度**: 膝盖相对于髋部的垂直高度
- **髋关节活动**: 髋关节弯曲角度
- **评分标准**: 抬腿高度 + 髋关节活动幅度

### 开合跳评估
- **手臂伸展**: 双臂展开幅度
- **腿部开合**: 双腿分开幅度
- **评分标准**: 手臂伸展 + 腿部开合

### 仰卧起坐评估
- **躯干角度**: 躯干与大腿之间的夹角
- **起身幅度**: 上半身抬起的高度
- **评分标准**: 起身幅度 + 动作标准度

### 平板支撑评估
- **身体挺直度**: 肩-髋-踝三点一线程度
- **髋部水平**: 髋部是否保持水平
- **评分标准**: 身体挺直度 + 稳定性

### 弓步蹲评估
- **前腿角度**: 前腿膝盖弯曲角度
- **后腿角度**: 后腿膝盖接近地面程度
- **评分标准**: 前后腿下蹲深度 + 平衡性

### 波比跳评估
- **动作阶段**: 站立→下蹲→平板→俯卧撑
- **连贯性**: 各阶段转换流畅度
- **评分标准**: 各阶段完成度 + 连贯性

## 🎓 教育应用场景

- **体育课堂**: 实时监测学生动作规范性
- **课后训练**: 学生自主练习时的动作指导
- **考试评估**: 体育考试动作的客观评分
- **教学反馈**: 为教师提供学生动作数据支持
